# Math Interactives — ELC #560

An interactive e-learning lesson built for [E-Learning Heroes Challenge #560](https://community.articulate.com/) — *Teaching Math Online with E-Learning Activities*.

**Live demo:** open `index.html` in any browser. No build step, no dependencies — one self-contained file.

## What's inside

A menu hub with three fully built guided activities, each following the same five-step arc: hook → guided discovery → payoff → deepen → free explore.

### Geometry · Unfold the Cube

A guided five-step activity that teaches faces, edges and vertices by letting the learner *experience* the ideas instead of memorizing them:

1. **Look closely** — spin a 3D cube by dragging and try to count its faces, running head-first into the "hidden faces" problem.
2. **The big reveal** — unfold the cube flat into its net, each color-coded face peeling away in sequence with a running count.
3. **Meet the net** — tap each of the six identical squares to check them off.
4. **Edges & corners** — fold it back up, then light up the 12 edges and 8 vertices (with a nod to Euler's formula).
5. **Your turn** — free exploration: scrub the fold slider, spin mid-fold, toggle the overlays.

The cube is pure CSS 3D (a hinged hierarchy of six faces driven by one fold-progress value) — works with mouse or touch.

### Fractions · Build the Pieces

Color-coded fraction bars where the numerator fills and the denominator cuts:

1. **Two numbers, two jobs** — which number does the cutting? (denominator vs numerator)
2. **Build it yourself** — use steppers to construct exactly 5/8.
3. **Same amount, new name** — adjust a bar cut into 8ths until it matches 1/2, discovering equivalence (1/2 = 4/8).
4. **Which is bigger?** — 2/3 vs 3/5, decided by sight instead of arithmetic.
5. **Your turn** — free play with both bars, a live "equivalent!" badge, and a pie-chart view toggle.

### Algebra · Balance the Scale

Solve for x on a literal balance scale with an animated beam, a mystery box, and unit blocks:

1. **The mystery box** — x + 3 = 7 sits balanced; guess the weight of x.
2. **Break it (on purpose)** — remove blocks from one side only and watch the scale tip and the equation turn red. The "=" sign is a promise.
3. **The fair way** — subtract 1 from *both* sides, three times, until the box opens: x = 4.
4. **Two mystery boxes** — 2x = 6 introduces dividing both sides in half.
5. **Your turn** — randomly generated equations, an operation trail (−1 / +1 / ÷2 color-coded chips), and a solve streak counter.
